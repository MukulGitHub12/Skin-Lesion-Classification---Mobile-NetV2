# 🩺 Skin Lesion Classification using Deep Learning

A Deep Learning based medical image classification project that detects and classifies different types of skin cancer using Convolutional Neural Networks (CNNs).

The project uses image processing and Deep Learning techniques to assist in the early detection of skin cancer through automated analysis of skin lesion images.

---

# 📌 Project Overview

Skin cancer is one of the most dangerous and rapidly increasing diseases worldwide. Early diagnosis and treatment are extremely important for improving survival rates.

This project focuses on building an intelligent Deep Learning model capable of classifying skin lesion images into different cancer categories with high accuracy.

The model is trained using medical image datasets and CNN architectures for efficient feature extraction and prediction.

# 📊 Dataset Information

The dataset contains dermoscopic images of various skin lesions.

The dataset used is HAM 10000 taken from Kaggle open source link

https://www.kaggle.com/datasets/kmader/skin-cancer-mnist-ham10000

### Dataset Classes Include:

* Melanoma
* Benign Keratosis
* Basal Cell Carcinoma
* Nevus
* Dermatofibroma
* Vascular Lesions

The images are preprocessed before training the Deep Learning model.
